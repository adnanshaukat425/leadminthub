## ğŸ”§ Customization

### Changing Colors
Edit the color variables in `styles.css`:
- Primary blue: `#1959d1`
- Background colors: `#ffffff`, `#f2f2f2`

### Updating Content
All content is in `index.html`. Simply edit the text within the respective sections.

### Chat Widget Configuration
The GoDaddy chat widget is configured in `index.html` before the closing `</body>` tag. To customize:
- Widget color: Edit `widgetColor` in the `_support['ui']['styles']` object
- Position: Change `position` in `_support['ui']['widget']`
- Notification text: Modify the `label.text` property
